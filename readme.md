## Prerequisites

### For fisrt exercise (1.py)

- **Python**: 3.7 or higher

Verify installation:
```bash
python --version
```

### For second exercise (Program.cs)

- **.NET SDK**: 6.0 or higher
- **Editor** (optional): Visual Studio Code or Visual Studio

Verify installation:
```bash
dotnet --version
```

If you don't have .NET installed:
- Download from: https://dotnet.microsoft.com/download

---

## Installation and execution

### Clone the repository

```bash
git clone https://github.com/Corosso/tech_test_Cadena.git
```

**Make sure when excuting the next comands to be on the project's root**


### Run Python Exercise:

```bash
python 1.py
```

### Run C# Exercise:

- First navigate to the C# project

```bash
cd p2
```

- Then run it
```bash
dotnet run
```
